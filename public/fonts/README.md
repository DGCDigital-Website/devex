# Neometric font files

Place licensed Neometric assets in this folder. The app loads them via `src/app/neometric-fonts.css` (URLs like `/fonts/Neometric-Regular.woff2`).

Required for the current `@font-face` set: `Neometric-Light`, `Regular`, `Medium`, `Bold`, `ExtraBold` (each `.woff2`, `.woff`, `.ttf` as shipped).
